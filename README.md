premièrement ceci est un project purement a but d'apprentissage personel

mon objectif avec ce projet est de faire un portage de mon jeu d'échec que j'ai déveloper en python (pour l'école) et d'en faire une version en pascal (le nouveau language que l'on utilise en cours) pour mieux comprendre le fonctionnement du language 

ce project utilise les librairie C : SDL2 , SDL2_ttf et SDL2_Image
avec les headers ( si le nom est bon ?) de https://github.com/PascalGameDevelopment/SDL2-for-Pascal

le seul fichier qu'il faut compiler est main.pas avec un compilateur pascal ( j'utilise personnellement free pascal compilator et je ne sais pas si tous marche :D )

pour le moment ce project est loin d'être completer et ne dispose que des déplacement des pièce sans fin de partie :)
